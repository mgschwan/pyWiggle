Pywiggle

A script to perform view interpolation between two stereo views

Examples

Sun (Nasa)
Commandline:
python stereointerpolate.py -r images/nasa.png  -a -m 20,10 -b 11,11 -s 5,5 -p 3,0 -S 8 -C 1 -P 1 -F test2_subpix.png -B 1


Beach
Commandline:
python stereointerpolate.py -r images/DSCF0640.LR.jpg  -a -m 20,10 -b 11,11 -s 5,5 -f 12 -p 12,0 -S 8 -C 1 -P 1 -F test2_subpix.png -B 1


Stone plate
Commandline:
python stereointerpolate.py -l images/stone1.bmp -r images/stone2.bmp  -m 20,10 -b 11,11 -s 5,5 -f 12 -p 12,0 -S 8 -C 1 -P 1 -F test2_subpix.png -B 1 --crop 15,15,436,353
